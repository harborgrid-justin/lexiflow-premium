   - Action: Check .env.example and configure

2. **AWS S3 Configuration**
   - Storage service supports S3 but requires credentials
   - Can fall back to local storage
   - Action: Configure STORAGE_PROVIDER=local for development

3. **Email Service**
   - SMTP configuration required for email sending
   - May fail if not configured
   - Action: Configure SMTP or use mail trap for testing

4. **WebSocket Package**
   - socket.io-client needs to be installed in frontend
   - Action: npm install socket.io-client

5. **TypeScript Strict Mode**
   - All code uses strict mode
   - Any type errors will block build
   - Action: Review TypeScript errors carefully

### Integration Warnings

   - Must run in correct order
   - Action: Use provided seed script
