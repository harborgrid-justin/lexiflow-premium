## ğŸ”„ Interaction Flows

### Edit Firm Profile
1. Click "Edit Profile"
2. Form fields become enabled
3. Make changes
4. Click "Save Changes" (shows spinner)
5. Success notification appears
6. Form returns to read-only mode

### Add New User
1. Click "Add User"
2. Modal opens
3. Fill in required fields
4. Click "Create User"
5. Validation checks
6. User added to table
7. Success notification
8. Invitation email sent

### IP Threat Check
1. Enter IP address in input field
2. Click "Verify" button
3. Bloom filter checks IP
4. Result card appears with color coding:
   - Green: Safe IP
   - Blue: Whitelisted IP
   - Red: Blocked IP
5. Shows detailed message

### Search & Filter Logs
1. Type in search box (real-time filtering)
2. Select status from dropdown
3. Table updates instantly
4. Click "Export" to download CSV
5. Empty state if no results
