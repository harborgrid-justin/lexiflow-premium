
- Active Cases
- Intake Pipeline

## Color Coding

**Theme-Aware Styling:**
- Primary Blue: Buttons, active states
- Emerald Green: Success messages
- Red: Error messages, required field indicators
- Amber: Warning messages
- Slate: Neutral backgrounds, borders

**Field Indicators:**
- Red asterisk (*): Required fields
- Gray background: View mode (read-only)
- White background: Edit mode (editable)

## Responsive Design

### Desktop (Wide Screen)
- 2-column grid for extracted fields
- Full-width text area (800px+)
- Side-by-side buttons

### Tablet (Medium Screen)
- 2-column grid maintained
- Slightly narrower text area (600-800px)
- Buttons stack on smaller tablets

### Mobile (Small Screen)
- 1-column grid for fields
- Full-width text area
- Stacked buttons
