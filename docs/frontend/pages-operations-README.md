# Operations Pages

## Overview
Firm business operations including billing, compliance, CRM, document repository, correspondence, and system administration.

## Pages (6)

### BillingPage
**Route**: `/operations/billing`  
**Purpose**: Time tracking, invoicing, and revenue management  
**Features**: Time entries, invoice generation, payment tracking, reports

### CompliancePage
**Route**: `/operations/compliance`  
**Purpose**: Ethics, conflicts, and regulatory compliance  
**Features**: Conflict checking, ethics rules, audit trails, certifications

### CRMPage
**Route**: `/operations/crm`  
**Purpose**: Client relationship management  
**Features**: Client database, interaction history, pipeline management

### DocumentsPage
**Route**: `/operations/documents`  
**Purpose**: Centralized document repository  
**Features**: Document management, version control, sharing, search

### CorrespondencePage
**Route**: `/operations/correspondence`  
**Purpose**: Email and communication management  
**Features**: Email integration, templates, tracking, archiving

### AdminPage
**Route**: `/admin`  
**Purpose**: System administration and settings  
**Features**: User management, permissions, system config, integrations

## Usage

```typescript
import { 
  BillingPage,
  CompliancePage,
  CRMPage,
  DocumentsPage,
  CorrespondencePage,
  AdminPage 
} from '@/components/pages/operations';
```

## Domain Scope
Business operations and firm management tools for administrators and office managers.
