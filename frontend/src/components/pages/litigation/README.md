# Litigation Pages

## Overview
Litigation-specific workflows including discovery, pleadings, evidence management, and strategy planning.

## Pages (4)

### DiscoveryPage
**Route**: `/litigation/discovery`  
**Purpose**: E-discovery management and document review  
**Features**: Document upload, OCR, review workflow, production sets

### PleadingsPage
**Route**: `/litigation/pleadings`  
**Purpose**: Legal document drafting and management  
**Features**: Pleading library, drafting tools, citation management

### EvidencePage
**Route**: `/litigation/evidence`  
**Purpose**: Evidence vault and chain of custody  
**Features**: Evidence tracking, chain of custody, metadata management

### LitigationStrategyPage
**Route**: `/litigation/strategy`  
**Purpose**: Visual strategy planning and case theory  
**Features**: Visual builder, timeline creation, argument mapping

## Usage

```typescript
import { 
  DiscoveryPage,
  PleadingsPage,
  EvidencePage,
  LitigationStrategyPage 
} from '@/components/pages/litigation';
```

## Domain Scope
Specialized tools for litigators handling complex civil and criminal cases.
