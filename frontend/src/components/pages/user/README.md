# User Pages

## Overview
User-specific pages including profile management and business development/marketing tools.

## Pages (2)

### ProfilePage
**Route**: `/profile`  
**Purpose**: User profile and account settings  
**Features**:
- Profile information
- Avatar upload
- Notification settings
- Preferences
- Security settings
- Activity history

### MarketingPage
**Route**: `/marketing`  
**Purpose**: Business development and marketing tools  
**Features**:
- Pipeline management
- Lead tracking
- Campaign management
- Analytics dashboard
- Contact management
- RFP tracking

## Usage

```typescript
import { 
  ProfilePage,
  MarketingPage 
} from '@/components/pages/user';
```

## Domain Scope
User management and business development tools for individual attorneys and marketing teams.
